# Library Management System

A desktop-based Library Management System developed using Java Swing and Object-Oriented Programming (OOP) concepts.

## Features

- User Authentication (Login / Signup)
- Book Management
- Search and Filtering
- Borrow and Return System
- Fine Calculation
- Transaction History
- GUI Dashboard
- Role-Based Access

## Technologies Used

- Java
- Java Swing
- OOP Principles
- Git & GitHub
